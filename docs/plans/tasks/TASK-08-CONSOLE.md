## 개요

콘솔 도구는 페이지의 console.log/warn/error 등을 캡처하고 분석하는 개발자 도구입니다.

### 주요 기능

1. **콘솔 인터셉트**: console.* 메서드 오버라이드
2. **로그 캡처**: 모든 콘솔 출력 수집
3. **스택 트레이스**: 에러 발생 위치 추적
4. **로그 필터링**: 레벨/검색/날짜 범위
5. **로그 내보내기**: JSON/TXT/CSV 포맷
6. **통계 대시보드**: 로그 집계 및 분석
