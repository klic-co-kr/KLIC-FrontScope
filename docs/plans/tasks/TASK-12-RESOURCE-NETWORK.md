## 개요

리소스 및 네트워크 최적화 도구는 다음 기능을 제공합니다:

1. **스토리지 클리너**: LocalStorage, SessionStorage, 쿠키 스캔 및 정리
2. **애니메이션 검사기**: CSS/JS 애니메이션 감지 및 성능 분석
3. **네트워크 모니터**: 네트워크 요청 실시간 모니터링 및 분석
4. **캐시 관리자**: 캐시 항목 분석 및 정리
